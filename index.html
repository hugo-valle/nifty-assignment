Nifty Assignment

Summary: In this assignment, students will create a Python program to calculate a water bill based on the number of gallons used in a household. The water bill will be determined using a tiered pricing structure, where different prices per gallon apply to different usage levels. The goal is to develop a program that calculates the total water bill for a household based on their water consumption.

Topics: This assignment threads consideration of sustainable water usage with basic programming concepts including variables, conditionals, branching, loops, functions, code refactoring and error handling, file I/O and testing.

Audience: The audience for this assignment is students in a CS1 course.

Length: The assignment has 7 tasks, each with a specific focus. In addition, it includes survey questions focusing on consciousness surrounding water consumption and conservation. It is expected that students can complete each task in one hour of less.

Strengths: This assignment is modular, so instructors may have students complete all seven tasks, or only a portion of them. We hope instructors will use the assignment to discuss sustainability concepts as well, although it works well without that component as well. It also helps students practice interacting with GitHub.

Weaknessess: The assignment is geared to courses that utilize Python only.

Dependencies: Basic familiarity with GitHub.

Variants: Instructors may select specific tasks to utilize and may opt in or out of the survey portion of the assignment.
